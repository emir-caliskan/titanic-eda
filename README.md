# Titanic EDA ve Hayatta Kalma Tahmini

## Proje Hakkında
Titanic yolcu verisi üzerinde keşifsel veri analizi (EDA) 
ve makine öğrenmesi modeli uygulaması.

## Kullanılan Teknolojiler
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Proje Adımları
1. Veri keşfi ve EDA
2. Görselleştirme
3. Veri ön işleme
4. Logistic Regression modeli
5. Model değerlendirmesi

## Sonuç
Model, test verisi üzerinde %80 doğruluk oranı elde etti.
Cinsiyetin ve yolcu sınıfının hayatta kalmada en belirleyici 
faktörler olduğu görüldü.
